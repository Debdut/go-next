# go-next

>  `go-next` is a template to create portable server apps in a single binary executable.

## Usage

```sh
# build
make

# build frontend
make build-next

# build the executable
make build-go

# run the binary
bin/go-next
```

## Notes

- Change binary name in the `Makefile`
- Change app name in `package.json` and `package-lock.json`
